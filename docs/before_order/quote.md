1. **_[訂單列表](https://cam.remotenc.com/order_list/) 頁面_**
    1. 找狀態為「等待估價」的訂單
2. **_訂單細節頁面_**
    2. 填入價格至尚無價格資料的訂購物品
    3. 選擇性填寫製程相關資料
        1. 如材料、邊界盒、表處、毛胚大小等資訊
            1. 若有材料相關資訊，可以自動生出叫料單
        2. 如外發商及進價相關資訊
3. **_修改訂單頁面 或 [訂單列表](https://cam.remotenc.com/order_list/) 頁面 或 該訂單細節頁面_**
    4. 完成後，將訂單狀態調整為「可報價」
